# [Klasy](https://www.typescriptlang.org/docs/handbook/classes.html) i [Interfejsy](https://www.typescriptlang.org/docs/handbook/interfaces.html)

W TypeScript mamy do dyspozycji klasy:
```ts
class X {}
```

interfejsy:
```ts
interface IX {}
```

klasy abstrakcyjne:
```ts
abstract class AC {
    abstract foo (): void;
}
```
